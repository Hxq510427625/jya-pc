# JYA

## 技术栈

- React 18
- React Router DOM
- Tailwind CSS
- Vite

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 项目结构

```
project/
├── src/
│   ├── components/
│   │   └── Navbar.jsx      # 导航栏组件
│   ├── pages/
│   │   ├── Home.jsx        # 主页面
│   │   ├── Projects.jsx    # 项目介绍页面
│   │   └── Contact.jsx     # 联系页面
│   ├── App.jsx             # 主应用组件
│   ├── main.jsx            # 入口文件
│   └── index.css           # 全局样式
├── index.html
├── package.json
├── tailwind.config.js
├── vite.config.js
└── postcss.config.js
```
